## Overview

Cosmos-Predict2.5 is latest branch of Cosmos World Foundation Models (WFMs) specialized for future state prediction built for physical AI, often referred to as world model. below are WFMs models present in Predict2.5.

| **Model** | **Workload** | **Use Case** |
|-----------|--------------|--------------|
| Cosmos Text2World | Inference | Text2World model is a text-to-video AI feature that generates a video directly from a written prompt |
| Cosmos Image2World | Inference | Image2World model allows you to create a video by using a text prompt + an input image. |
| Cosmos Video2World | Inference | Video2World model generates video from a combination of a text prompt and an input video |

Lets see some examples of above models with their details such as prompts, inputs, etc.
